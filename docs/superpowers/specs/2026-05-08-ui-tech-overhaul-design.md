# 520 网站 UI 技术全面升级 · 设计规格

## 目标

将网站从"深空毛玻璃+极光霓虹"升级为全 Three.js 统一渲染的炫酷技术流体验。
6 层效果在同一渲染管线中协同工作，所有效果统一、性能可控。

## 方案

**全 Three.js 统一方案**：一个 Three.js 实例管理所有 3D 效果，EffectComposer 串联 Bloom 和转场后处理，DOM 层仅保留文字内容和交互 UI。

## 第 1 层 · 银河粒子旋涡 + 极光 + 交互

替代现有三层静态星空（~2000 粒子）。

### 螺旋结构
- 2-4 条对数螺旋臂，30-50K 粒子
- 核心暖金色（`#e0b860`），向旋臂外围渐变青蓝（`#50d0b0`）→ 紫（`#b890e0`）
- 粒子大小 0.5-4px，核心密集亮大，旋臂稀疏，尘埃层微小
- 单 BufferGeometry + PointsMaterial，CanvasTexture 圆形精灵

### 极光飘带
- 3-5 条极光飘带沿螺旋臂内侧流动
- 复用现有 aurora curtain 的 GLSL shader 技术，改造为飘带状 mesh
- 颜色青绿→紫罗兰渐变，与旋臂色彩呼应
- 鼠标移动时飘带偏移速度比银河快（更"轻"）

### 极光粒子雾
- 1000-2000 个较大半透明粒子散布在旋臂间隙
- 缓慢 hue shift 色彩流动，模拟极光弥漫感

### 交互
- **鼠标引力场**：光标附近粒子被轻微吸引（最大偏移 ~15px），距离越近偏移越大
- **点击脉冲波**：点击产生同心圆能量波，推动周围粒子先外推再回弹
- **悬停亮区**：光标附近粒子亮度提升 2-3 倍，颜色向暖金偏移，离开后恢复

### 动态
- 缓慢自转（~0.05°/帧）+ 鼠标视差偏移（±8°）+ 粒子独立微漂移

## 第 2 层 · Bloom 后处理

- EffectComposer + UnrealBloomPass
- 参数：threshold 0.6 / strength 1.5 / radius 0.4
- 影响：银河核心、极光飘带、3D 文字、流星尾巴等高亮元素
- 暗色粒子（亮度低于阈值）不受影响
- 从 CDN 导入（与现有 Three.js import map 方式一致）

## 第 3 层 · 3D Hero 场景

首屏 Hero 区域增加 3D 视觉元素，DOM 文字内容（打字机、锁屏按钮）保留。

### 3D 粒子标题
- Canvas 绘制"Ray的小屋"→ 采样像素 → 3D 空间排列粒子
- 带 Bloom 后发出辉光
- 位置在 Hero 区域上方

### 3D 照片水晶
- 现有 5 张图钉照片改为 3D 漂浮相框
- TexturedPlane mesh，散落在 Hero 区域四周不同 z 深度
- 各自缓慢自转，鼠标靠近时微微倾斜

## 第 4 层 · 深度视差 3D 滚动

滚动驱动相机 z 轴位置微移（±50 单位），各层按 z 深度获得不同视差速度：
- 银河背景：0.3x（最远）
- 3D Hero 元素：0.6x
- 内容卡片：1.0x（基准）
- 浮尘/萤火虫：1.5x（最近）

## 第 5 层 · Shader 电影转场

自定义 ShaderPass 实现，叠加在 EffectComposer 最后一级，uniform progress 0→1 驱动。

三种转场效果：
- **Warp（星云穿越）**：粒子从中心飞驰，800ms，用于解锁/导航
- **Dissolve（粒子化散）**：画面碎裂成粒子飘散再重组，1000ms，段落间切换
- **Glitch（数字失真）**：RGB 通道分离+画面撕裂，600ms，快速切换

## 第 6 层 · 高级微交互

### 卡片 3D Tilt
- 鼠标在卡片上的位置映射 rotateX/rotateY（±8°）
- CSS transform 实现，`perspective` 父容器

### 点击能量波
- Canvas 叠加层，点击处扩散同心圆涟漪，400ms 淡出

### 悬停粒子爆发
- 鼠标悬停可交互元素时，10-20 个小粒子从边缘喷射，向上飘散淡出
- Canvas 2D 或 Three.js 粒子实现

## 实施原则

- 分步实施：每层独立完成，验证后再做下一层
- 保留现有功能：音乐系统、许愿星空板、日记卡片、3D 轮播、灯箱等功能不变
- 禁止修改：图片 JPG、音乐 MP3、文字内容
- 移动端：粒子数减半，效果参数降低
- 不写注释，代码简洁

## 技术依赖

- Three.js 0.160.0（已引入）
- EffectComposer、RenderPass、UnrealBloomPass、ShaderPass（新增 CDN import）
